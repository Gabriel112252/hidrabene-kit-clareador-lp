# Kit Clareador Facial — Landing Page

Protótipo de landing page (mobile-first, responsiva) do **Kit Clareador Facial** da Hidrabene Derma — o Protocolo Clareador Intensivo: Sabonete Líquido + Sérum Multicorretivo + Protetor Solar FPS 70 Clareador.

> ⚠️ **Conteúdo confidencial.** Página não indexada (`noindex`). Compartilhe o link apenas internamente.

## Ver online
Hospedado via GitHub Pages — link na seção **Deployments / Pages** do repositório.

## Rodar local
Arquivo estático: basta abrir `index.html` no navegador.

## Estrutura
- `index.html` — página completa (HTML + CSS + JS inline)
- `assets/` — imagens, vídeos (`video/`), logo e selos da marca (`brand/`)

## Pendências conhecidas
- **Mockups da oferta**: os 3 cards de kit usam `kit-packshot.jpg` como placeholder até chegarem os mockups definitivos.
- **Preços**: os cards estão com `R$ XX,XX` (aguardando valores).
- **Checkout**: a constante `CHECKOUT_URL` no JS está vazia (passa `?kit=` por kit quando configurada).
